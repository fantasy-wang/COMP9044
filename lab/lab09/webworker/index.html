Web Workers

This exercise builds from the fetch and list exercises, but rather than making our fetch requests on the main thread (which blocks) we're going to delegate that process to a Web Worker. We're gong to use the CatAPI this time, which, well delivers us cats. Our Web Worker will do the fetching of our image data from the CatAPI:

https://api.thecatapi.com/v1/images/search?&mime_types=image/gif

Then, we're going to combine a setInterval() call every 5s to move between each image in the output div. This is a super pared down example of how real-world long polling might work.
